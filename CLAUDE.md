# Project Instructions

## ARIS Skills
This project uses ARIS skills for automated research workflows.
Skills are managed by `install_aris.sh` — do not edit `.claude/skills/` manually.

## Available Workflows
- `/research-lit` — Literature search
- `/idea-discovery` — Idea discovery pipeline
- `/experiment-bridge` — Implement and run experiments
- `/auto-review-loop` — Automated review loop
- `/paper-writing` — Paper writing pipeline
- `/research-wiki` — Persistent research memory
