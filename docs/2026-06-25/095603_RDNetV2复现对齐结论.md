# RDNetV2复现对齐结论

结论：`strategies/semi_rdnet_v2.py` 不是 `/home/guo/project/other_method/SSL/2025CVPR_RDNet` 的逐行复现。排除 backbone 后，它保留了 RDNet 的 RGB 分支、depth 分支、EMA、DPA、跨模态伪标签更新、跨模态对比/MSE/L2 这条主干；主要差异来自公平实验协议统一、`image_s` 输入缺失、输出语义和评估口径。

## 核心证据

| 维度 | 当前实现 | 原 RDNet | 判断 |
|---|---|---|---|
| 强增强输入 | `semi_rdnet_v2.py:116` 直接读取 `batch_data["image_s"]` | `datasets/kvasir_SEG.py:122-131` 生成 ColorJitter+blur 后的 `image_s` | 当前 `data/transforms.py` 没有生成 `image_s`，默认不可跑 |
| 输出语义 | `semi_rdnet_v2.py:58-60` 要求 2 类 logits 后 softmax 取前景 | `models/ResUNet.py:171-174` 返回 1 通道 sigmoid 概率图 | 非 backbone 差异，影响所有 loss 和验证 |
| 监督标签 | `semi_rdnet_v2.py:62-63` 将 label 转二值 `[B,1,H,W]` | 原数据集 `ToTensor()` 得到 `[B,1,H,W]` 浮点 mask | 基本可对齐 |
| DPA | `semi_rdnet_v2.py:80-112` 复写 DPA，按 iter/max_iterations 调度 | `utils/rd_loss.py:15-80` 按 epoch/nEpoch 调度 | 逻辑接近，但调度粒度不同 |
| 跨模态伪标签更新 | `semi_rdnet_v2.py:146` 执行 `update_pseudo_labels` | `train.py:226-231` 调用 `update_pseudo_labels` 后进入 depth-aware consistency | RDNet 核心机制，按复现主干保留 |
| loss 公式 | `semi_rdnet_v2.py:142-164` 权重和组合基本复刻 | `train.py:220-242` 原公式 | 公式骨架接近，输入概率语义不同 |
| 优化器 | `semi_rdnet_v2.py:34-46` SGD, momentum 0.9, weight_decay 1e-5 | `train.py:152-159` SGD + poly LambdaLR | 形式接近 |
| 默认训练超参 | `core/args.py:117,146-163` lr=1e-4, max_iterations=30000, AMP/grad_clip/early_stopping 默认开 | `train.sh:15-19` lr=1e-3, total_iter=40000，无 AMP/clip/early stopping | 默认实验协议不一致 |
| 数据增强/尺寸 | `data/transforms.py:119-143` 224、rot_flip 或 ±20 rotate | `datasets/kvasir_SEG.py:47-53` 320、H/V flip、90° rotation、zoom，另有 ColorJitter+blur | 差异大 |
| 验证口径 | `core/train.py:356` 对 2 类 logits softmax+argmax | 原模型输出 1 通道 sigmoid | 非同口径 |

## 风险分级

| 等级 | 问题 | 最小修复方向 |
|---|---|---|
| P0 | 默认数据管线不产生 `image_s`，策略训练入口会 KeyError | 在现有 `RandomGenerator` 内按原 RDNet 生成强增强 `image_s` |
| P0 | 当前使用 2 类 logits+softmax，原 RDNet 是 1 通道 sigmoid 概率图 | 明确选择：要么改成 1 通道 RDNet 口径，要么承认是工程适配版 |
| P1 | 训练默认协议不同：lr、iter、resize、AMP、grad_clip、early stopping | RDNet 复现实验脚本显式覆盖原始超参 |
| P1 | 增强链不一致，缺少原 strong augmentation | 补齐 Resize 320、H/V flip、90° rotation、zoom、ColorJitter、blur |
