## 损失函数说明

| 缩写 | 全称 | 说明 |
|------|------|------|
| CE | Cross Entropy | 交叉熵损失 |
| Dice | Dice Loss | Dice系数损失 |
| MSE | Mean Squared Error | 均方误差（一致性约束） |
| NT-Xent | Normalized Temperature-scaled Cross Entropy | 温度归一化对比损失 |
| InfoNCE | Information Noise-Contrastive Estimation | 信息噪声对比估计 |
| SSIM | Structural Similarity Index | 结构相似性损失 |
| L1 | L1 Loss | 绝对值损失 |
| MI | Mutual Information | 互信息（解耦损失） |

## 关键组件说明

| 组件 | 说明 |
|------|------|
| EMA Teacher | 指数移动平均教师模型，参数由student通过EMA更新 |
| Depth Teacher | 可训练的深度教师模型，独立于EMA，通过反向传播更新 |
| Appearance Teacher | 外观教师模型，使用RGB为中心的扰动训练，用于几何/外观解耦 |
| Learnable_Prototypes | 可学习的类别原型库，用于原型对比学习 |
| Depth Guider | 深度引导模块，在编码器各层生成空间注意力对RGB特征做门控增强 |
| Geometry/Appearance Projector | 几何/外观投影头，用于特征解耦 |

## 版本演进关系

```

├── MT-DepthGuider-v1/v2/v3 (深度引导增强)
│   └── MT-DepthGuider-Proto-v1 (加原型对比)
│       └── MT-DepthGuider-Proto-Teacher-v1 (加depth teacher)
│           └── MT-DepthGuider-Proto-Teacher-v2 (depth teacher弱辅助化)
│               └── MT-DepthGuider-Proto-Teacher-v3 (加appearance teacher + 解耦)

├── MeanTeacher-Contrast-v1 (对比学习分支)
├── MeanTeacher-Text-v1 (文本特征对齐)
├── Proto-v1 (纯原型对比)
└── OnlyDepthInput (仅深度输入)
```
